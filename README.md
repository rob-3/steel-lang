# Steel

Steel is an in-development programming language designed for ease of use.

## Installation

To install Steel, run the following commands.

```
git clone https://github.com/rob-3/steel-lang.git
cd steel-lang
npm install
```

## Getting Started

To use the REPL run:
```
npm start
```

To run a file use:
```
npm start -- file.stl
```
